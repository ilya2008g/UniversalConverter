# UniversalConverter

Кроссплатформенное приложение для перевода чисел между системами счисления с произвольными основаниями (2–500).  
Поддерживает периодические дроби, составные цифры в квадратных скобках (`[123]`) и чтение входных данных из файла.

## Возможности

- Перевод целых и дробных чисел (включая периодические) между любыми основаниями от 2 до 500.
- Ввод цифр как одиночными символами (`0-9`, `A-Z`, `a-z`), так и в квадратных скобках для оснований >36.
- Проверка корректности ввода с подробными сообщениями об ошибках.
- Загрузка данных из текстового файла (формат: три строки – `p`, `q`, `input`).
- Сохранение результата в файл по желанию.
- Классы длинной арифметики (`BigInteger`, `BigFraction`) с алгоритмом Карацубы и бинарным поиском для деления.
